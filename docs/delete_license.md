# Deleting License


In Veeam Service Provider Console Administrator Portal, you can delete licenses from client and hosted Veeam Backup & Replication and Veeam ONE servers. You can also revoke licenses from managed Veeam backup agent.

In This Section

* [Deleting Veeam Backup & Replication License](delete_vbr_license.md)
* [Deleting Veeam ONE License](delete_one_license.md)
* [Revoking Veeam Backup Agent Licenses](revoke_licenses.md)
