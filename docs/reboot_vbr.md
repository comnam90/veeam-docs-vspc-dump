# Restarting Backup Services


In some cases, you may need to restart backup services on backup servers, for example after installation of patches and updates on Veeam Backup & Replication servers.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Portal Administrator, Site Administrator, Portal Operator.

Restarting Backup Services

To restart backup services on Veeam Backup & Replication server:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Discovery.
2. Open the Backup Servers tab.
3. Select the necessary backup servers in the list.
4. At the top of the list, click Server Actions and choose Restart Backup Services.

Alternatively, you can right-click the necessary backup server, choose Server Actions and select Restart Backup Services.

1. In the displayed window, click Yes to confirm restart of the backup services.
