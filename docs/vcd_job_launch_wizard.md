# Step 1. Launch New Cloud Director Backup Job Wizard


To launch the New Cloud Director Backup Job wizard:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Backup Jobs.
2. Open the Virtual Machines tab and navigate to Virtual Infrastructure.
3. At the top of the jobs list, click New > Cloud Director Backup Job.

Alternatively, you can right-click on the jobs list and select New > Cloud Director Backup Job.

Veeam Service Provider Console will open the Create Job window.

1. Select the server on which you want to create the job and click Create.

For servers running Veeam Backup & Replication version 13.0.1 or later: Veeam Service Provider Console will open the New Backup Job wizard from the Veeam Backup & Replication Web UI (for Portal Administrator) or the New Cloud Director Backup Job wizard (for Site Administrator, Portal Operator).

[For Portal Administrator] Follow the procedure as described in the Veeam Backup & Replication User Guide, starting from [Step 2. Specify Job Name and Description](https://helpcenter.veeam.com/docs/vbr/userguide/backup_job_name_vm_web.html).

[For Site Administrator, Portal Operator] Create a backup job as described in this procedure.

For servers running Veeam Backup & Replication version 13 or earlier: Veeam Service Provider Console will open the New Cloud Director Backup Job wizard . Create a backup job as described in this procedure.
