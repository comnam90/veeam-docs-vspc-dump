# Trends


The Trends dashboard collection provides information on usage trends in managed and hosted infrastructures as well as backup and cloud services consumption trends.

Required Privileges

To perform this task, a user must have the following role assigned: Portal Administrator.

Accessing Dashboards

To access the dashboards:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Trends.
2. Use tabs at the top to open the necessary dashboard view:

* [Infrastructure](trends_infrastructure.md)
* [Backup Services](trends_backup.md)
* [Cloud Connect](trends_cc.md)

1. To show data for a specific Veeam Cloud Connect site, reseller and company, use the sites and reseller/company filters at the top left corner of the Veeam Service Provider Console window.
