# TOPs


The TOPs dashboard collection provides information about top resource consumers among client companies and managed backup and cloud infrastructures.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Portal Administrator, Site Administrator, Portal Operator, Read-only User.

Accessing TOPs Dashboards

To access the dashboards:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click TOPs.
2. Use tabs at the top to open the necessary dashboard view:

* [Companies](top_tenants.md)
* [Infrastructure](top_infrastructure.md)

1. To show data for a specific Veeam Cloud Connect site, reseller and company, use the sites and reseller/company filters at the top left corner of the Veeam Service Provider Console window.
