# Enabling ConnectWise Manage Integration


To allow Veeam Service Provider Console to communicate with ConnectWise Manage, you must configure application plugin and enable integration features.

Enabling ConnectWise Manage Integration

To enable integration with ConnectWise Manage, perform the following steps:

1. [Obtain ConnectWise Manage API keys](cwm_obtain_keys.md).
2. [Configure ConnectWise Manage plugin connection](cwm_connect_plugin.md).
3. [Enable integration features](cwm_enable_features.md).
