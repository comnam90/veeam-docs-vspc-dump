# Updating Veeam Service Provider Console License


When your Veeam Service Provider Console license expires, you must update it.

You can use the following methods to update the license:

* [Manual license update](update_license_manually.md)
* [Automatic license update](update_license_automatically.md)

Both methods require that Veeam Service Provider Console has Internet connection. During the license update, Veeam Service Provider Console connects to the Veeam License Update Server on the Internet, downloads a new Veeam Service Provider Console license from it (if the license is available), and installs it to replace the old license.

|  |
| --- |
| Tip: |
| You can also update the license in the Veeam License Management Portal and install it in Veeam Service Provider Console to replace the expired license. However, this method is not covered in this guide. |
