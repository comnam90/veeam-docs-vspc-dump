# Downloading Logs


To troubleshoot issues with managed Veeam products or Veeam Service Provider Console you may need to gather diagnostic logs, or view task and activity logs.

* Diagnostic logs are required when you submit tickets to Veeam Customer Technical Support.
* Task and activity logs keep record of the most crucial operations that Veeam Service Provider Console management agents perform in the client managed infrastructure, as well as configuration, management and security-related activities.

In Veeam Service Provider Console, you can gather the following logs:

* [Logs from the Veeam Cloud Connect server](download_sp_server_logs.md)
* [Logs from managed Veeam Backup Agent](download_agent_job_logs.md)
* [Logs from managed Veeam Backup & Replication servers](download_client_server_logs.md)
* [Logs from managed Veeam ONE servers](one_server_logs.md)
* [Logs from managed Veeam Backup for Microsoft 365 servers](vbo_server_logs.md)
* [Veeam Service Provider Console logs](download_product_logs.md)
* [Veeam Service Provider Console management agent logs](download_vac_agent_logs.md)
* [ConnectWise Manage plugin logs](cwm_logs.md)
* [VCSP Pulse plugin logs](pulse_logs.md)
* [Task and event logs](view_task_activity_data.md)
