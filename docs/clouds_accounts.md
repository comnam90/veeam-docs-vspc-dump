# Managing Veeam Backup for Public Clouds Accounts


In plugin, you can manage Amazon Web Services and Microsoft Azure accounts, database administrator accounts and guest OS accounts used to connect and manage Veeam Backup for Public Clouds appliances.

You can manage accounts created for your hosted appliances and accounts created by your client companies. Note that you cannot manage accounts created as part of appliance deployment.
