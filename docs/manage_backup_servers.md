# Managing Veeam Backup & Replication Servers


In Veeam Service Provider Console, you can install Veeam Backup & Replication on client and hosted computers, connect existing Veeam Backup & Replication servers to Veeam Service Provider Console, patch and upgrade Veeam Backup & Replication servers in client and hosted infrastructures, reset Veeam Backup & Replication server installation ID and access Veeam Backup & Replication server PowerShell console.

|  |
| --- |
| Note: |
| For Linux Veeam Backup & Replication servers, in Veeam Service Provider Console you can only assign custom tags. For other management tasks, use Veeam Host Management Console. For details, see section [Performing Maintenance Tasks](https://helpcenter.veeam.com/docs/vbr/userguide/hmc_perform_maintenance_tasks.html?ver=13) of the Veeam Backup & Replication User Guide. |
