# Modifying Reseller Cloud Resources


To modify cloud resources allocated to a reseller, perform the following steps:

1. [Launch Edit Cloud Resources wizard](launch_cloud_reseller_wizard.md).
2. [Specify site scope](reseller_site_scope.md).
3. [Allocate cloud resources](allocate_cloud_reseller_quota.md).
4. [Configure bandwidth settings](configure_bandwidth_reseller.md).
5. [Review settings](review_cloud_reseller_settings.md).
