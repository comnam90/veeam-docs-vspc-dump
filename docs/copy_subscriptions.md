# Copying Subscription Plans


Instead of creating a new subscription plan from scratch, you can create a copy of an existing subscription plan and modify its settings. This can be useful if you need to create a set of similar plans with only a few settings that differ.

Required Privileges

To perform this task, a user must have the following role assigned: Portal Administrator.

Copying Subscription Plan

To create a copy of a subscription plan:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the configuration menu on the left, click Templates and navigate to the Subscription Plans tab.
3. Select the necessary subscription plan in the list.
4. At the top of the list, click Copy.

Alternatively, you can right-click the necessary subscription plan and choose Copy.

1. At the Plan Info step of the wizard, change the name and description of a subscription plan copy.
2. Modify the necessary subscription plan settings as described in [Creating Subscription Plans](create_subscriptions.md).
3. Save changes.
