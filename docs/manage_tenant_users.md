# Managing Company Users


By default, the only user that can access the Client Portal on behalf of a company is the Company Owner. To allow other employees in a managed company to work with the Veeam Service Provider Console Client Portal, you can create new portal users with the necessary user permissions.

Required Privileges

To perform the task, a user must have one of the following roles assigned: Portal Administrator, Site Administrator, Portal Operator.
