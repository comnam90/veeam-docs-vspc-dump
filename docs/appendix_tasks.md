# Task Reference


This section provides reference of all Veeam Service Provider Console tasks.

Task Reference

| Name | Description |
| Backup agent activation status change | Activates or deactivates backup agent. |
| Backup agent CBT driver deployment | Installs CBT driver on the backup agent. |
| Backup agent CBT driver removal | Deletes CBT driver from the backup agent. |
| Backup agent deployment | Installs backup agents to the remote computers. |
| Backup agent job change | Changes backup agent job settings. |
| Backup agent job start | Starts backup agent job. |
| Backup agent job stop | Stops backup agent job. |
| Backup agent patch deployment | Installs private hotfixes and patches for the backup agent. |
| Backup agent removal | Deletes backup agent from the remote computer. |
| Backup agent service restart | Restarts Veeam backup agent service. |
| Backup agent settings change | Changes application settings for the backup agent. |
| Backup agent UI mode change | Switches the UI operation mode for the backup agent. |
| Backup agent upgrade | Upgrades backup agent version. |
| Backup job removal | Removes Veeam backup agent job configuration. |
| Backup repository re-configuration | Creates individual folders for each backup agent pointed to the CIFS repository. |
| Backup server deployment | Deploys Veeam Backup & Replication server. |
| Backup server installation files download | Manages Veeam Backup & Replication installation files download on the remote computer. |
| Backup server job action | Performs a management action on a Veeam Backup & Replication job. |
| Backup server patch deployment | Installs private hotfixes and patches for the Veeam Backup & Replication server. |
| Backup server upgrade | Upgrades Veeam Backup & Replication server version. |
| Cloud gateway list update | Updates the list of available cloud gateways to Veeam Service Provider Console management agent. |
| Computer reboot | Reboots remote computer operating system. |
| Data collection job | Collects data from managed Veeam products. |
| Failover plan execution | Initiates VM failover plan. |
| Files download | Manages support logs download tasks from the management agent. |
| Files upload | Manages files upload tasks on the management agent. |
| Job agent scheduler | Starts scheduled tasks on the management agent. |
| Management agent connection attempt | Connects management agent to the backup portal. |
| Management agent connection settings update | Updates management agent connection settings. |
| Management agent master role creation | Sets master role to the management agent. |
| Management agent master role removal | Removes master role from the management agent. |
| Management agent patch deployment | Installs private hotfixes and patches for the management agent. |
| Management agent removal | Deletes management agent from the remote computer. |
| Management agent task start | Sends user tasks to the management agent. |
| Management agent update | Updates management agent version. |
| Management agent version check | Verifies management agent version. |
| New backup server ID creation | Creates a new ID for the backup server. |
| New Veeam ONE ID creation | Creates a new ID for the Veeam ONE server. |
| Remote computer discovery | Discovers computers in the remote networks. |
| Tasks queue clean up | Cleans up the tasks queue on the management agent. |
