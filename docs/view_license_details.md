# Viewing License Details


In Veeam Service Provider Console, you can obtain consolidated license usage data for:

* Veeam backup agents managed in Veeam Service Provider Console
* Veeam Backup & Replication servers managed in Veeam Service Provider Console
* Veeam ONE servers managed in Veeam Service Provider Console
* Veeam Backup for Microsoft 365 servers managed in Veeam Service Provider Console
* Veeam Cloud Connect servers connected to Veeam Service Provider Console
* Backups stored on cloud repositories and VM replicas registered on cloud hosts in Veeam Cloud Connect that comes as part of Veeam Service Provider Console

To view Veeam Service Provider Console license details:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the menu on the left, click License Information.
3. Depending on license details you want to view, do one of the following:

* To view details on the license that is currently installed in Veeam Service Provider Console, on the Overview tab, click the License Status link.

For details, see [Veeam Service Provider Console License Key Details](license_details.md).

* To view cumulative information about license usage for Veeam Service Provider Console, Veeam Cloud Connect and Veeam Backup & Replication, open the Overview tab.

For details, see [Overview](license_usage.md).

* To view information about license usage for Veeam Cloud Connect servers connected to Veeam Service Provider Console, open the Veeam Cloud Connect tab.

For details, see [Veeam Cloud Connect](cc_license.md).

* To view information about license usage for Veeam Backup & Replication servers managed in Veeam Service Provider Console, open the Veeam Backup & Replication tab.

For details, see [Veeam Backup & Replication](backup_servers_license_status.md).

* To view information about license usage for Veeam backup agents managed in Veeam Service Provider Console, open the Veeam Agent tab.

For details, see [Veeam Agent](backup_agents_license_status.md).

* To view information about license usage for Veeam ONE servers managed in Veeam Service Provider Console, open the Veeam ONE tab.

For details, see [Veeam ONE](one_servers_license_status.md).

* To view information about license usage for Veeam Backup for Microsoft 365 servers managed in Veeam Service Provider Console, open the Veeam Backup for Microsoft 365 tab.

For details, see [Veeam Backup for Microsoft 365](vbo_servers_license_status.md).
