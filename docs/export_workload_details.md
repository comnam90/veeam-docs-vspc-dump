# Viewing and Exporting Protected Workload Details


You can view details on workloads protected by managed backup servers and export them to a CSV or XML file. Details are available for the following workloads:

* [Virtual Machines](export_protected_vms.md)
* [File Shares](export_protected_file_shares.md)
* [Object Storage](export_protected_object_storage_data.md)
* [Computers Protected by Veeam Backup & Replication](export_protected_computers_vbr.md)

Required Privileges

To perform this task, a user must have one of the following roles assigned: Portal Administrator, Site Administrator, Portal Operator, Read-only User.
