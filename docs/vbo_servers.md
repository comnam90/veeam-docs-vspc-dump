# Managing Veeam Backup for Microsoft 365 Servers


In plugin, you can add hosted Veeam Backup for Microsoft 365 servers which you want to manage and for which you want to monitor license usage.

Note that for Veeam Backup for Microsoft 365 version 6 you can only monitor license usage.

To add client Veeam Backup for Microsoft 365 servers, you must deploy Veeam Service Provider Console management agents on client computers where these servers are installed. For details, see [Deploying Management Agents Manually](deploy_management_agents.md).
