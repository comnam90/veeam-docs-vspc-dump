# License Types


Veeam Service Provider Console accepts instance-based or merged Veeam Backup & Replication licenses for managing Veeam backup agents.

Commercial Licenses

There are two types of commercial licenses available for Veeam Service Provider Console:

* Rental license is a full license intended for service providers. The expiration date of the Rental license is set to the contract end date, which is the last day of the month and normally 1 month from the contract start date.

You must be registered with the Veeam® Cloud & Service Provider (VCSP) program to work with Veeam Service Provider Console for Service Providers.

* Subscription license is a full license intended for enterprises that use Veeam Cloud Connect for internal data protection operations. The expiration date of the Subscription license is set to the end of the subscription term. The Subscription license term is normally 1-5 years from the license issue date.

This license type requires that the customer has either a Microsoft Enterprise Agreement (EA) or VMware Enterprise License Agreement (ELA). To be eligible, customers are required to provide a valid Microsoft EA number or VMware ELA number.

Free Licenses

There are two types of free licenses available for Veeam Service Provider Console:

* Evaluation license is a full license that can be used for product evaluation. The evaluation license is valid for 30 days from the moment of the product download.
* NFR license is a full license that can be used for product demonstration, training and education. This license is not for resell or commercial use.

These license types require submitting usage report to Veeam every week. If license ID is provided for these licenses, automatic updated is enabled by default and cannot be disabled.
