# Step 1. Launch Edit Cloud Resources Wizard


To launch the Edit Cloud Resources wizard:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the configuration menu on the left, click Catalog.
3. Click the Veeam Cloud Connect plugin tile.
4. In the menu on the left, click Resellers.
5. At the top of the list, click Edit Cloud Resources.
