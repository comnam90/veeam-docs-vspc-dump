# Integration with Veeam Cloud Connect


Veeam Service Provider Console offers integration with Veeam Cloud Connect. Integration allows you to allocate Veeam Cloud Connect tenants and resources to client companies and resellers and allow companies to store Veeam Backup & Replication backups and replicas on cloud repositories.

Integration offers the following capabilities:

* Configuring Veeam Cloud Connect resources. You can create Veeam Cloud Connect tenants and allocate to the tenants Veeam Cloud Connect backup and replication resources.
* Assigning Veeam Cloud Connect tenants. You can assign cloud tenants to client companies to provide them Veeam Cloud Connect resources allocated to cloud tenants.
* Managing reseller Veeam Cloud Connect resources. You can allocate Veeam Cloud Connect backup and replication resources to managed resellers.

This solution is intended for service providers who:

* Use Veeam Service Provider Console to manage and monitor license usage data.
* Manage client companies who want to store their data in the cloud.
* Use Veeam Backup & Replication to back up and restore virtual infrastructure workloads.

This section assumes that you have a good understanding of Veeam Backup & Replication and Veeam Cloud Connect.

Before You Begin

To allocate Veeam Cloud Connect resources to resellers and client companies, you must connect Veeam Cloud Connect servers to Veeam Service Provider Console. For details, see [Connecting Veeam Cloud Connect Servers](connect_cloud_server.md).
