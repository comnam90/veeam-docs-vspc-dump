# Managing Veeam ONE Servers


In plugin, you can connect hosted Veeam ONE servers for which you want to monitor license usage.

To add client Veeam ONE servers, you must deploy Veeam Service Provider Console management agents on client computers where these servers are installed. For details, see [Deploying Management Agents Manually](deploy_management_agents.md).
