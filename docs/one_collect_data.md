# Collecting Data from Veeam ONE Servers


By default, the data between Veeam Service Provider Console and Veeam ONE is synchronized every 5 minutes.

To synchronize data immediately, you can initiate data collection from hosted Veeam ONE servers manually:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the configuration menu on the left, click Catalog.
3. Click the Veeam ONE plugin tile.
4. In the menu on the left, click Servers.
5. Select the necessary server in the list.
6. Click Collect Data.

To collect data from client Veeam ONE servers, go to the Discovery > ONE Servers tab. For details, see [Collecting Data](collect_agent_data.md#one).
