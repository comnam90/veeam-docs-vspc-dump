# Creating Cloud Tenants


To register a cloud tenant account in Veeam Service Provider Console, you can do either of the following:

* [Create a tenant in Veeam Service Provider Console](create_tenant_in_vspc.md)
* [Create a tenant in Veeam Cloud Connect](create_tenant_in_cloud.md)

Cloud tenant data is synchronized between Veeam Service Provider Console and Veeam Cloud Connect: when you create a cloud tenant in Veeam Service Provider Console, this tenant appears in Veeam Cloud Connect, and conversely.
