# Managing Hosted Veeam Backup & Replication Servers


In plugin, you can connect hosted Veeam Backup & Replication and Veeam Backup Enterprise Manager servers, collect data from hosted Veeam Backup & Replication and Veeam Backup Enterprise Manager servers, assign custom tags to hosted Veeam Backup & Replication servers, view and export hosted Veeam Backup & Replication and Veeam Backup Enterprise Manager server details.

Note that in plugin you can manage only the servers hosting Veeam Backup & Replication version 12.1 or later. For details on managing older Veeam Backup & Replication servers, see [Managing Veeam Backup & Replication Servers](manage_backup_servers.md).
