# Deleting Backup Reports


You can delete generated backup reports if you no longer need them. These can be report configurations created in the Veeam Service Provider Console Reseller Portal and Administrator Portal.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Portal Administrator, Site Administrator, Portal Operator.

Deleting Backup Reports

To delete created backup report:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Reports.
2. Select the necessary reports in the list.
3. Click Remove.

Alternatively, you can right-click the necessary report and choose Remove.

1. In the displayed window, click Yes to confirm removal.
