# Configuring Veeam Service Provider Console Settings


To start working with Veeam Service Provider Console, you must perform the following tasks for its initial configuration:

1. [Configure security certificates](install_certificate.md).

To secure connections between the Veeam Cloud Connect server, Veeam Service Provider Console server, managed Veeam Backup & Replication servers, computers running Veeam backup agents and Veeam Service Provider Console Web UI component, configure security certificates.

1. [Fill your company profile](fill_company_profile.md).

The company profile contains information about the service provider company, such as the company name, address and so on. Company profile details will be displayed in email notifications sent by Veeam Service Provider Console, in invoices and backup reports.

1. [Customize portal branding](customize_branding.md).

You can customize Veeam Service Provider Console appearance: choose a color scheme, change the default portal logo, report logo, website icon and website address.

1. [Configure notification settings](configure_email_settings.md).

To send out invoices to client companies, and receive alarm notifications by email, you must configure SMTP server settings, global notification settings and billing notification settings.
