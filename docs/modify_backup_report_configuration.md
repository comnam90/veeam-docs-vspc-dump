# Modifying Backup Report Configuration


You can modify settings of a saved backup report configuration.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Portal Administrator, Site Administrator, Portal Operator.

Modifying Backup Report Configuration

To modify a report configuration:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Reports.
2. Open the Configurations tab.
3. Select the necessary report configuration and click Edit at the top of the list.

Alternatively, you can right-click the necessary report configuration and choose Edit.

1. Modify settings of the report configuration as described in [Configuring Backup Reports](configure_backup_reports.md).

You can modify all report configuration settings except the report type and company (for individual report).

1. Save changes.
