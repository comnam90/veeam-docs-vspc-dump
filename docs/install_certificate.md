# Configuring Security Certificates


You can configure the following Veeam Service Provider Console security certificates:

* [Server certificate](configure_server_certificate.md)
* [Web UI certificate](configure_web_ui_certificate.md)

To perform this task, a user must have the following role assigned: Portal Administrator.

|  |
| --- |
| Note: |
| Update Veeam Service Provider Console Server and Veeam Service Provider Console Web UI certificates regularly. For details on certificates update, contact your Certificate Authority.  If a Veeam Service Provider Console Server certificate is expired, connection with management agents will not be interrupted, but a new certificate must be configured shortly. |

For more information about all certificates used in the Veeam Service Provider Console infrastructure, see section [Veeam Service Provider Console Certificates](https://helpcenter.veeam.com/docs/vac/deployment/vspc_certificates.html?ver=9.2) in the Deployment Guide.
