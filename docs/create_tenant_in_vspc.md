# Creating Cloud Tenants in Veeam Service Provider Console


To create a new cloud tenant in Veeam Service Provider Console, perform the following steps:

1. [Launch the New Cloud Tenant wizard](launch_cloud_tenant_wizard.md).
2. [Select cloud tenant type](specify_tenant_type.md).
3. [Specify cloud tenant credentials](specify_tenant_credentials.md).
4. [Allocate cloud resources](allocate_cloud_resources.md).
5. [Configure bandwidth settings](configure_bandwidth.md).
6. [Review cloud tenant settings](review_cloud_tenant_settings.md).
