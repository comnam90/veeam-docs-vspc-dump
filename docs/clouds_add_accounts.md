# Adding Veeam Backup for Public Clouds Accounts


In plugin, you can add the following accounts for Veeam Backup for Public Clouds appliances:

* [Amazon Web Services](clouds_aws_accounts.md)
* [Microsoft Azure](clouds_azure_accounts.md)
* [Google Cloud](clouds_google_accounts.md)
* [Databases](clouds_database_accounts.md)
* [Guest OS](clouds_guest_accounts.md)

Alternatively, you can create the necessary accounts during appliance deployment. For details on adding appliances, see the following topics:

* [Amazon Web Services](clouds_add_aws.md)
* [Microsoft Azure](clouds_add_azure.md)
* [Google Cloud](clouds_add_google.md)
