# Modifying Veeam Backup for Microsoft 365 Jobs


To modify Veeam Backup for Microsoft 365 job settings:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Backup Jobs.
2. Open the Microsoft 365 Objects tab.
3. Choose the necessary job in the list and click Edit.

Alternatively, you can right-click the necessary job and choose Edit.

1. In the Edit Job wizard, modify Veeam Backup for Microsoft 365 job settings.

For details on backup job settings, see [Creating Veeam Backup for Microsoft 365 Jobs](create_vbo_jobs.md).

1. Save changes.
