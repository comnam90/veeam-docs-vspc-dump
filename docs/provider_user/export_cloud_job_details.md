# Viewing and Exporting Job Details


You can view details on Veeam Backup for Public Clouds jobs configured on managed backup servers and export them to a CSV or XML file. Job details are available for the following workloads:

* [Virtual Machines](export_cloud_vm_job_details.md)
* [File Shares](export_cloud_file_share_jobs.md)
* [Cloud Databases](export_databases.md)
* [Cloud Networks](export_networks.md)

Required Privileges

To perform this task, a user must have one of the following roles assigned: Company Owner, Company Administrator, Location Administrator, Location User.
