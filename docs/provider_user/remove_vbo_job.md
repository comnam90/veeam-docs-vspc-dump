# Removing Veeam Backup for Microsoft 365 Jobs


If you no longer want to protect data of a managed Veeam Backup for Microsoft 365 organization, you can remove a backup job configuration.

To remove a job configuration from Veeam Backup for Microsoft 365:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Backup Jobs.
2. Open the Microsoft 365 Objects tab.
3. Choose the necessary jobs in the list and click Remove.

Alternatively, you can right-click the necessary job and choose Remove.

1. In the Remove Job window, click Yes to confirm removal.
