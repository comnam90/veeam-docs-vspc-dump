# Sending Backup Reports


You can manually send backup reports.

Before you send a backup report, make sure that all recipients specified in the report configuration have an email address provided in their user profiles. For details on specifying an email address for company users, see [Modifying Portal Users](modify_users.md). For details on configuring your email address, see [Modifying User Profile](modify_user_profile.md).

Required Privileges

To perform this task, a user must have one of the following roles assigned: Company Owner, Company Administrator, Location Administrator.

Sending Backup Reports

To send one or more backup reports manually:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Reports.
2. Select the necessary reports in the list.
3. Click Send.

Alternatively, you can right-click the necessary report and choose Send.
