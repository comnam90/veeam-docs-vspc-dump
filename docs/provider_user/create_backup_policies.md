# Creating Windows Backup Policies


You can create custom backup policies with Veeam Agent for Microsoft Windows job settings that will suit your data protection requirements for managed computers.

Before you create a backup policy, check prerequisites described in [Before You Begin](template_configuration_prerequisites.md).

To create a backup policy, complete the following steps:

1. [Launch the New Backup Policy wizard](launch_template_wizard.md).
2. [Specify the job template name and description](specify_template_name.md).
3. [Choose the backup agent operation mode](choose_operation_mode.md).
4. [Choose the backup mode](choose_backup_mode.md).
5. [Choose volumes to back up](choose_volumes.md).
6. [Choose folders to back up](choose_files.md).
7. [Choose backup destination](choose_backup_destination.md).
8. [Specify local storage settings](specify_storage_settings.md).
9. [Specify shared folder settings](specify_folder_settings.md).
10. [Specify backup server settings](specify_backup_server_settings.md).
11. [Specify backup repository settings](specify_backup_repository_settings.md).
12. [Specify cloud backup settings](specify_cloud_backup_settings.md).
13. [Specify cloud repository quota](specify_cloud_quota.md).
14. [Specify advanced job settings](specify_advanced_job_settings.md).
15. [Choose backup cache location](choose_cache_location.md).
16. [Specify guest OS processing options](specify_processing_options.md).
17. [Configure backup schedule](configure_backup_schedule.md).
18. [Review backup job settings](review_backup_job_settings.md).
