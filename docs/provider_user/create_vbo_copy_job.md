# Creating Backup Copy Jobs


You can create backup copy jobs for existing Veeam Backup for Microsoft 365 backup jobs.

Before you create a backup copy job, check prerequisites described in [Before You Begin](vbo_copy_job_configuration_prerequisites.md).

To create a backup copy job, complete the following steps:

1. [Launch the New Backup Copy Job wizard](vbo_copy_job_launch_wizard.md).
2. [Choose the backup copy target](vbo_copy_job_target.md).
3. [Configure the backup copy schedule](vbo_copy_job_schedule.md).
4. [Review the backup copy job settings](vbo_copy_job_summary.md).
