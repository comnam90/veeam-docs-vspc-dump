# Restoring Linux Files


You can restore individual files and folders from backups created with Veeam Agent for Linux.

Before you perform restore, check prerequisites described in [Before You Begin](flr_prerequisites_lin.md).

To restore backed up files and folders, perform the following steps:

1. [Open the file-level restore portal](flr_open_flr_wizard_lin.md).
2. [Select a restore point and specify objects to restore](flr_select_restore_point_lin.md).
3. [Perform the restore action](flr_perform_restore_lin.md).
4. [Review restored objects and restore session details](flr_audit_lin.md).
