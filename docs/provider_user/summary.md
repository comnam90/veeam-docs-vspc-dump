# Monitoring


Veeam Service Provider Console includes a set of monitoring dashboards that provide information about workloads protected with Veeam products, as well as details on the amount of cloud resources consumed by the company.

* [Overview](overview.md)

The dashboard consolidates information about your company, including information about active alarms and so on. This dashboard presents the big picture and serves as the starting point from which you can drill down to dashboards and views with further details.

* [RPO & SLA](protected_data_summary.md)

The dashboard provides information on protected workloads in your backup and cloud infrastructures.

* [Resources](resources.md)

The dashboard provides information on cloud resources consumed by your company.

* [Session States](job_overview.md)

The dashboard provides information about job session statuses and data protection efficiency.
