# Deleting API Keys


If you no longer use an API key for integration with third party solutions, you can remove created API key. Note that after you remove an API key, Veeam Service Provider Console will break all connections made with the removed key.

Required Privileges

To perform this task, a user must have Company Owner, Company Administrator.

Deleting API Keys

To delete an API key:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the configuration menu on the left, click REST API Keys.
3. Select the necessary API key and click Remove at the top of the list.

Alternatively, you can right-click the necessary API key and choose Remove.

1. In the Delete API Key window, click Yes.
