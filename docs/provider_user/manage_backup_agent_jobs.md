# Managing Veeam Backup Agent Jobs


You can manage Veeam backup agent jobs configured on managed computers. Job management operations include starting and stopping backup jobs, disabling and enabling backup jobs and deleting jobs.

To access job management, you must have at least one Veeam backup agent in your infrastructure.
