# Managing Veeam Service Provider Console Agents


To manage Veeam backup agents, Veeam ONE, Veeam Backup for Microsoft 365 and Veeam Backup & Replication servers in Veeam Service Provider Console and perform discovery of computers in managed infrastructure, you must install Veeam Service Provider Console management agents on machines where Veeam products are deployed. Veeam Service Provider Console management agents can collect data, perform management and configuration tasks on managed machines or perform discovery in managed infrastructure. For details on management agent roles and functions, see section [Architecture in Brief](https://helpcenter.veeam.com/docs/vac/deployment/architecture.html) of the Deployment Guide.

| --- |
| Note: |
| Integration with Veeam Service Provider Console is available for Veeam Backup for Microsoft 365 version 6 or later and Veeam ONE version 11a or later. |

* [Restart management agent service](restart_management_agent.md)
* [Collect data from machines hosting management agents](collect_agent_data.md)
