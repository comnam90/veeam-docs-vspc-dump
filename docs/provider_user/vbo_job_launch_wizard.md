# Step 1. Launch New Backup Job Wizard


To launch the New Backup Job wizard:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Backup Jobs.
2. Open the Microsoft 365 Objects tab.
3. At the top of the jobs list, click Create Job and select Backup Job.

Veeam Service Provider Console will open the New Backup Job wizard.
