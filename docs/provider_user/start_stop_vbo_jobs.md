# Starting and Stopping Jobs


You can start and stop Veeam Backup for Microsoft 365 jobs without accessing the Veeam Backup for Microsoft 365 console on managed servers. For example, if a backup job has failed, you can start this job again in Veeam Service Provider Console to avoid company data loss.

Starting Jobs

To start Veeam Backup for Microsoft 365 jobs:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Backup Jobs.
2. Open the Microsoft 365 Objects tab.
3. Select the necessary job in the list and click Actions > Start.

Alternatively, you can right-click the necessary job and choose Actions > Start.

Stopping Jobs

To stop Veeam Backup for Microsoft 365 jobs:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Backup Jobs.
2. Open the Microsoft 365 Objects tab.
3. Select the necessary job in the list and click Actions > Stop.

Alternatively, you can right-click the necessary job and choose Actions > Stop.
