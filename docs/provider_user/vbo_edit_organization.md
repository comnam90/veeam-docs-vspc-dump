# Modifying Microsoft 365 Organizations


To modify registered Microsoft 365 organization:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the configuration menu on the left, click Catalog.
3. Click the Veeam Backup for Microsoft 365 plugin tile.
4. In the menu on the left, click Organizations.

1. Select the necessary organization from the list.
2. At the top of the list, click Edit.

Alternatively, you can right-click the necessary organization and choose Edit.

Veeam Service Provider Console will launch the Edit Organization wizard.

1. Modify organization settings as described in [Registering Microsoft 365 Organizations](vbo_register_organization.md).
2. Save changes.
