# Managing Veeam ONE


Veeam Service Provider Console allows you to manage Veeam ONE servers located in your infrastructure and monitor alarms triggered on these servers.

To manage Veeam ONE servers, you must complete the following steps:

1. [Deploy Veeam Service Provider Console management agents](deploy_management_agents.md) on computers hosting Veeam ONE servers.

1. [Enable alarms synchronization for managed Veeam ONE servers](enable_disable_alarm_sync.md).

You can enable alarms data collection for connected Veeam ONE servers to manage in Veeam Service Provider Console alarms triggered on these servers.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Company Owner, Company Administrator, Company Tenant, Location Administrator, Location User.

Location Users can only view and export Veeam ONE server details.
