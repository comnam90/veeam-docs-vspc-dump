# Managing Veeam Backup for Public Clouds Accounts


In plugin, you can manage Amazon Web Services and Microsoft Azure accounts, database administrator accounts used to back up and restore Veeam Backup for Public Clouds workloads.

You can manage accounts created for your hosted appliances. Note that you cannot manage accounts created as part of appliance deployment.
