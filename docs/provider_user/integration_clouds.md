# Integration with Veeam Backup for Public Clouds


Veeam Service Provider Console offers integration with Veeam Backup for Amazon Web Services, Veeam Backup for Microsoft Azure and Veeam Backup for Google Cloud to monitor and manage Veeam Backup for Public Clouds jobs in Veeam Service Provider Console.

The integration is available if your service provider has enabled the integration for your company. You can connect to Veeam Service Provider Console a Veeam Backup & Replication server with connected Veeam Backup for Public Clouds appliances or you can ask your service provider to assign Veeam Backup for Public Clouds appliances to your company.

Integration offers the following capabilities:

* Jobs management. You can manage Veeam Backup for Public Clouds backup jobs from Veeam Service Provider Console.

* Consolidated monitoring. You can consolidate information about resources consumed by your Veeam Backup for Public Clouds appliances in Veeam Service Provider Console reports.

This solution is intended for company owners and company administrators who:

* Use Veeam Service Provider Console to manage and monitor data protection operations.

* Use Veeam Backup for Public Clouds to back up and restore Amazon Web Services, Microsoft Azure and Google Cloud workloads.

This section assumes you have a good understanding of Veeam Backup for Amazon Web Services, Veeam Backup for Microsoft Azure and Veeam Backup for Google Cloud.
