# Creating Linux Backup Policies


You can create custom backup policies with Veeam Agent for Linux job settings that will suit data protection requirements for managed computers.

Before you create a backup policy, check prerequisites described in [Before You Begin](template_configuration_prerequisites_lin.md).

To create a backup policy, complete the following steps:

1. [Launch the New Backup Policy wizard](launch_template_wizard_lin.md).
2. [Specify the backup policy name and description](specify_template_name_lin.md).
3. [Choose the backup agent operation mode](choose_operation_mode_lin.md).
4. [Choose the backup mode](choose_backup_mode_lin.md).
5. [Choose volumes to back up](choose_volumes_lin.md).
6. [Choose directories to back up](choose_files_lin.md).
7. [Choose backup destination](choose_backup_destination_lin.md).
8. [Specify local storage settings](specify_storage_settings_lin.md).
9. [Specify shared folder settings](specify_folder_settings_lin.md).
10. [Specify backup server settings](specify_backup_server_settings_lin.md).
11. [Specify backup repository settings](specify_backup_repository_settings_lin.md).
12. [Specify cloud backup settings](specify_cloud_backup_settings_lin.md).
13. [Specify cloud repository quota](specify_cloud_quota_lin.md).
14. [Specify advanced job settings](specify_advanced_job_settings_lin.md).
15. [Specify guest OS processing options](specify_processing_options_lin.md).
16. [Configure backup schedule](configure_backup_schedule_lin.md).
17. [Review backup job settings](review_backup_job_settings_lin.md).
