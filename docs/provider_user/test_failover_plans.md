# Testing Failover Plans


You can test cloud failover plans without accessing the Veeam Backup & Replication console on a managed backup server.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Company Owner, Company Administrator, Company Tenant, Location Administrator.

Testing Failover Plans

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Failover Plans.
2. At the top of the list, in the Type section on the filter bar, choose the Cloud option to show all cloud failover plans.
3. Select one or more plans in the list.
4. At the top of the list, click Test.

Alternatively, you can right-click the necessary failover plan and choose Test.

Veeam Service Provider Console will display the result of the test in the Status column.
