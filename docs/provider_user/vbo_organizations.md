# Managing Microsoft 365 Organizations


In plugin, you can add Microsoft 365 organizations for which you want to back up and restore data.

You can register Microsoft 365 organizations on your own Veeam Backup for Microsoft 365 servers or on servers assigned to you by your service provider. If you want to add and manage Microsoft 365 organizations on service provider servers, make sure your service provider has assigned Veeam Backup for Microsoft 365 resources to your company. You can see Veeam Backup for Microsoft 365 servers assigned to your company on the Getting Started plugin page.
