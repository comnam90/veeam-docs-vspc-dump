# Creating Veeam Backup for Microsoft 365 Jobs


In Veeam Service Provider Console, you can configure Veeam Backup for Microsoft 365 job settings on managed servers and servers assigned to you by your service provider.

You can configure the following job types:

* [Backup Jobs](create_vbo_backup_job.md)

A backup job defines a list of users, groups, sites and teams to back up, a repository where to store backups, a schedule according to which new backups must be created.

* [Backup Copy Jobs](create_vbo_copy_job.md)

To enhance data protection and ensure compliance with the 3-2-1 rule, you can extend your backup jobs with backup copy capabilities. Backup copy allows you to create an instance of your backed-up data in a cloud-based object storage.
