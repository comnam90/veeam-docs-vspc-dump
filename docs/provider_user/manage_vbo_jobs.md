# Managing Veeam Backup for Microsoft 365 Jobs


You can manage Veeam Backup for Microsoft 365 backup and backup copy jobs configured on managed servers.

|  |
| --- |
| Note: |
| Veeam Backup for Microsoft 365 job management is available for Veeam Backup for Microsoft 365 version 7 or later. |

You can modify jobs, remove jobs, start and stop jobs, enable and disable jobs, view and export job details.
