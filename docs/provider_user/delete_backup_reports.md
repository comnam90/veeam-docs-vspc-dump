# Deleting Backup Reports


You can delete generated backup reports if you no longer need them. These can be reports created both in the Veeam Service Provider Console Client Portal, and created by your service provider.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Company Owner, Company Administrator, Location Administrator.

Deleting Backup Reports

To delete one or more generated backup reports:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Reports.
2. Select the necessary reports in the list.
3. Click Remove.

Alternatively, you can right-click the necessary report and choose Remove.

1. In the displayed window, click Yes to confirm removal.
