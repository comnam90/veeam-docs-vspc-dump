# Creating Backup Jobs


You can create backup jobs that will suit data protection requirements for managed Microsoft 365 organizations.

Before you create a backup job, check prerequisites described in [Before You Begin](vbo_job_configuration_prerequisites.md).

To create a backup job, complete the following steps:

1. [Launch the New Backup Job wizard](vbo_job_launch_wizard.md).
2. [Specify the backup job name and description](vbo_job_name_description.md).
3. [Choose organization to back up](vbo_job_organization.md).
4. [Choose the backup mode](vbo_job_backup_mode.md).
5. [Choose backup repository](vbo_job_backup_repository.md).
6. [Configure backup schedule](vbo_job_schedule.md).
7. [Review the backup job settings](vbo_job_summary.md).
