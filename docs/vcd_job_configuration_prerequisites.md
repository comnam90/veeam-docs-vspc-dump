# Before You Begin


Before you configure a backup job:

* Check prerequisites specified in the [Creating Backup Jobs for Cloud Director VMs](https://helpcenter.veeam.com/docs/vbr/userguide/vcloud_director_perform_backup.html#considerations-and-limitations) section of the Veeam Backup & Replication User Guide.
* Allocate Veeam Backup & Replication resources to the company for which you plan to create a backup job. For details, see [Step 5. Allocate Service Quota](allocate_hosted_quota.md).
* Map VMware Cloud Director organizations to the company for which you plan to create a backup job. For details, see [Mapping VMware Cloud Director Organizations](vbr_map_vcd.md).
