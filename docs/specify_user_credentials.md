# Step 3. Specify User Credentials


At the User Info step of the wizard, specify the following settings:

1. In the Username, Password and Confirm password fields, specify user credentials of a Company Owner.

The client will use these credentials to access the Veeam Service Provider Console Client Portal. For details on the Company Owner, see [Managing Company Owners](manage_company_owners.md).

1. To allow company users to utilize Veeam Service Provider Console REST API capabilities, select the Enable access to REST API of the backup portal check box.
