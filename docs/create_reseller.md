# Creating Resellers


You can register a new reseller account in Veeam Service Provider Console and allocate necessary resources to the created reseller.

Required Privileges

To perform this task, a user must have the following role assigned: Portal Administrator.

Creating Resellers

To create a new reseller in Veeam Service Provider Console, perform the following steps:

1. [Launch the New Reseller wizard](launch_reseller_wizard.md).
2. [Specify reseller details](specify_reseller_details.md).
3. [Specify user credentials](specify_reseller_credentials.md).
4. [Configure services](configure_reseller_services.md).
5. [Configure multi-factor authentication](mfa_reseller.md).
6. [Customize welcome email](reseller_welcome_email_custom.md).
7. [Review reseller settings](review_reseller_settings.md).
8. [Provide connection details to reseller](do_next_reseller.md).
9. [Assign companies to manage](add_reseller_companies.md).
