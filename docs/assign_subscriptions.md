# Assigning Subscription Plans


To charge a company for provided services, you must assign a subscription plan to this company.

Required Privileges

To perform this task, a user must have one of the following roles assigned: Portal Administrator, Site Administrator, Portal Operator.

Assigning Subscription Plan

You can assign a subscription plan when registering a company account in Veeam Service Provider Console. For details, see [Creating Companies](create_companies.md).

Alternatively, you can assign a subscription plan to one or more companies as follows:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Companies.
2. Select one or more companies in the list.
3. At the top of the list, click Manage and select Set Billing.

Alternatively, you can right-click the necessary company, choose Manage and select Set Billing.

1. In the Set Billing window, choose the necessary subscription plan.

[For Portal Administrator] You can select an existing subscription plan from the list, or click Create New to create a new subscription plan. For details on creating subscription plans, see [Creating Subscription Plans](create_subscriptions.md).

1. Click Apply.
