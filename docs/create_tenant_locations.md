# Step 9. Create Company Locations


If a company has more than one office or business unit, you can create multiple locations for this company in Veeam Service Provider Console. With multiple locations configured for a company, you will be able to differentiate services and cloud resources consumed by each office.

For details on working with company locations, see [Managing Locations](manage_locations.md).
