# Configuring Linux Discovery Rules


For Linux computers Veeam Service Provider Console supports the following discovery methods:

* [Network-based discovery](configure_network_discovery_linux.md)

This method allows you to discover computers based on a range of IP addresses.

* [Import-based discovery](configure_import_discovery_linux.md)

This method allows you to discover computers by importing a list of IP addresses or DNS names from a CSV file.
