# Connecting Veeam Backup & Replication Servers


To allow Veeam Service Provider Console to communicate with a hosted Veeam Backup & Replication or Veeam Backup Enterprise Manager server, you must configure a connection to this server.

When you connect a hosted Veeam Backup & Replication or Veeam Backup Enterprise Manager server to Veeam Service Provider Console, Veeam Service Provider Console deploys its management agent on the server. The management agent is responsible for transmitting commands from Veeam Service Provider Console to Veeam Backup & Replication or Veeam Backup Enterprise Manager server, performing management operations, collecting data from Veeam Backup & Replication and Veeam Backup Enterprise Manager and communicating it back to Veeam Service Provider Console.
