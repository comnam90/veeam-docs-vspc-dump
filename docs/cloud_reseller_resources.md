# Managing Reseller Cloud Resources


In the Veeam Service Provider Console plugin, you can modify cloud resources allocated to resellers as well as view and export reseller details.

Note that if you assign to a reseller a company that has cloud resources allocated, these resources will be automatically allocated to the reseller.

Before You Begin

To modify Veeam Cloud Connect resources, you must enable cloud services for the reseller. For details, see [Enable Cloud Services](enable_reseller_cloud.md).
