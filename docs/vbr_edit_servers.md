# Modifying Veeam Backup & Replication Servers


You can modify settings of the hosted Veeam Backup & Replication and Veeam Backup Enterprise Manager servers:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the configuration menu on the left, click Catalog.
3. Click the Veeam Backup & Replication plugin tile.
4. In the menu on the left, click Infrastructure.
5. Select the necessary server in the list.
6. At the top of list, click Edit.
7. Modify Veeam Backup & Replication or Veeam Backup Enterprise Manager server settings specified in [Connecting Veeam Backup & Replication Servers](vbr_connect_servers.md).
8. Save changes.
