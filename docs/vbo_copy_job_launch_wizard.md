# Step 1. Launch New Backup Copy Job Wizard


To launch the New Backup Copy Job wizard:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. In the menu on the left, click Backup Jobs.
2. Open the Microsoft 365 Objects tab.
3. Select the necessary backup job in the list.
4. At the top of the jobs list, click Create Job and select Backup Copy Job.

Veeam Service Provider Console will open the New Backup Copy Job wizard.
