# Before You Begin


Before you configure a backup job:

* Check prerequisites specified in the [Considerations and Limitations](https://helpcenter.veeam.com/docs/vbr/userguide/backup_job_limitations.html) section of the Veeam Backup & Replication User Guide.
* Allocate Veeam Backup & Replication resources to the company for which you plan to create a backup job. For details, see [Step 5. Allocate Service Quota](allocate_hosted_quota.md).
* Assign VMware vSphere tag to the company for which you plan to create a backup job. For details, see [Assigning Tags to Companies](vbr_assign_vsphere_tag.md).
