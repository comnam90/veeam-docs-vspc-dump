# Managing License


Veeam Service Provider Console allows you to view license details, install and update licenses for the following products:

* Veeam Service Provider Console
* Veeam Cloud Connect
* Veeam Backup & Replication
* Veeam ONE
* Veeam Backup for Microsoft 365
