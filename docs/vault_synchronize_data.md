# Synchronizing Data


After you configure plugin connection, the data between Veeam Service Provider Console and Veeam Data Cloud Vault will be synchronized every 10 minutes.

To synchronize the necessary data immediately, you can refresh it manually:

1. Log in to Veeam Service Provider Console.

For details, see [Accessing Veeam Service Provider Console](access_vac.md).

1. At the top right corner of the Veeam Service Provider Console window, click Configuration.
2. In the configuration menu on the left, click Catalog.
3. Click the Veeam Vault plugin tile.
4. In the menu on the left, click Veeam Data Cloud and navigate to the necessary tab, or click Backup Servers.
5. At the top of the page, click Sync Now.

Plugin will synchronize all data between Veeam Service Provider Console and Veeam Data Cloud.
