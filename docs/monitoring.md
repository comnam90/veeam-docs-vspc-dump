# Monitoring


Veeam Service Provider Console includes a set of monitoring dashboards and views that provide information about managed companies, consumed cloud resources and the state of backup infrastructures on the service provider and client sides.

* [Overview](overview.md)

The dashboard consolidates information about managed companies, including information about active alarms, Veeam products health state, and so on. This dashboard presents the big picture and serves as the starting point from which you can drill down to dashboards and views with further details.

* [RPO & SLA](protected_data_summary.md)

The dashboard provides information on protected workloads in managed backup and cloud infrastructures.

* [Resources & Billing](summary_dashboard.md)

The dashboard provides information on revenue, invoices and utilized cloud backup and replication resources.

* [Session States](job_overview.md)

The dashboard provides information about job session statuses and data protection efficiency.

* [TOPs](tops.md)

The dashboard collection provides information about top resource consumers among client companies and managed backup and cloud infrastructures.

* [Trends](trends.md)

The dashboard collection provides information on managed and hosted infrastructure, backup and cloud services usage trends.
