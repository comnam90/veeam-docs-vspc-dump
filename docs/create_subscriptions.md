# Creating Subscription Plans


Out of the box, Veeam Service Provider Console comes with a preconfigured Default subscription plan. This subscription plan does not include any charge rates, and is intended for trial and testing purposes. You can customize this subscription plan to suit your needs, or create new subscription plans if you need more than one.

Required Privileges

To perform this task, a user must have the following role assigned: Portal Administrator.

Creating Subscription Plan

To create a new subscription plan, perform the following steps:

1. [Launch the New Subscription Plan wizard](launch_subscription_wizard.md).
2. [Specify a name and description of the subscription plan](specify_name_description.md).
3. [Specify currency, tax and discount](specify_currency.md).
4. [Specify rates for general services](specify_service_fees.md).
5. [Specify rates for managed backup services](specify_backup_rates.md).
6. [Specify rates for Cloud Connect services](specify_cloud_backup_rates.md).
7. [Specify rates for Microsoft 365 services](specify_microsoft_backup_rates.md).
8. [Specify rates for public cloud services](specify_public_cloud_rates.md).
9. [Specify rates for rental licenses](specify_license_rates.md).
10. [Optional] Specify rates for external plugins.

For details on external plugins, see [Integration with Custom Plugins](integration_custom.md).

1. [Review subscription plan settings](review_subscription_settings.md).
