# Creating Mac Backup Policies


You can create custom backup policies with Veeam Agent for Mac job settings that will suit data protection requirements for managed computers.

Before you create a backup policy, check prerequisites described in [Before You Begin](template_configuration_prerequisites_mac.md).

To create a backup policy, complete the following steps:

1. [Launch the New Backup Policy wizard](launch_template_wizard_mac.md).
2. [Select the backup policy access type](policy_access_type_mac.md).
3. [Specify the backup policy name and description](specify_template_name_mac.md).
4. [Choose the backup agent operation mode](choose_operation_mode_mac.md).
5. [Choose folders to back up](choose_files_mac.md).
6. [Choose backup destination](choose_backup_destination_mac.md).
7. [Specify local storage settings](specify_storage_settings_mac.md).
8. [Specify shared folder settings](specify_folder_settings_mac.md).
9. [Specify backup server settings](specify_backup_server_settings_mac.md).
10. [Specify backup repository settings](specify_backup_repository_settings_mac.md).
11. [Specify cloud backup settings](specify_cloud_backup_settings_mac.md).
12. [Specify cloud repository quota](specify_cloud_quota_mac.md).
13. [Specify advanced job settings](specify_advanced_job_settings_mac.md).
14. [Configure backup schedule](configure_backup_schedule_mac.md).
15. [Review backup policy settings](review_backup_job_settings_mac.md).
